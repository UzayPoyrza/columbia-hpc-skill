# Terremoto HPC Cluster User Documentation

Source: https://columbiauniversity.atlassian.net/wiki/spaces/rcs/pages/62140769/Terremoto+HPC+Cluster+User+Documentation

---

**Please Note**: This documentation is a work in progress. If you require information that has not yet been included please email [hpc-support@columbia.edu](mailto:hpc-support@columbia.edu) and a member of the support team will assist you.

The Terremoto High Performance Computing (HPC) Cluster is a service supporting research at Columbia.

All cluster inquiries should be directed to [hpc-support@columbia.edu](mailto:hpc-support@columbia.edu). Please include the following information in your email:

1. Your Columbia [UNI](https://cuit.columbia.edu/cuit/manage-my-uni).
2. The name of the cluster you are working on ("Terremoto" in this case).
3. Job ID numbers, if asking about a job issue.

All computing at Columbia must abide by the [Columbia University E-Mail & Computing Policies](http://www.columbia.edu/cu/policy/).
