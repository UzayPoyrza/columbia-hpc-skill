# Welcome to Free User Documentation

The Free High Performance Computing (HPC) Cluster is a service supporting research at Columbia.

**SUPPORT:** Please note that for FREE USER ACCOUNTS, support is limited to ONLINE DOCUMENTATION only.

All other cluster inquiries, including EDU accounts, should be directed to [hpc-support@columbia.edu](mailto:hpc-support@columbia.edu). Please include the following information in your email:

1. **Your UNI** (e.g., ab1234)
2. **Cluster Name:** "Ginsburg"
3. **Job ID Numbers:** (If reporting a specific job failure or error)

### **Usage Policy**

All computing activities on Insomnia must comply with the [Columbia University E-Mail & Computing Policies](https://universitypolicies.columbia.edu/).

- **Sharing:** Users are prohibited from sharing UNI credentials.
- **Data:** Strictly no HIPAA or Sensitive Data is permitted on the shared cluster.

**Please Note**: This documentation is a work in progress. If you require information that has not yet been included please email [hpc-support@columbia.edu](mailto:hpc-support@columbia.edu) and a member of the support team will assist you.
